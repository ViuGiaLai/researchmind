v0.6 Features

Built for PhD students, scholars, and systematic review teams — powered by Tauri + FastAPI.

Import & Index

PDF, DOCX, EPUB, BibTeX, RIS, Zotero. Status queue, OCR for scanned PDFs, bge-m3 embedding + ChromaDB.

Hybrid Search

BM25 (SQLite FTS5) + vector semantic + cross-encoder rerank. Saved search, filter by collection/tag/year.

RAG Chat

Streaming, multi-paper, Evidence-only mode, citation panel, DOI verification, trust panel.

Review & Screening

Insights, screening board, 7-section review builder, comparison matrix, export DOCX/Markdown.

Evidence Matrix

Extract claims by criterion, sourced quotes, visual comparison table across papers.

Help Center

Docs, FAQ, keyboard shortcuts, release notes — ? button in the corner, keeps the sidebar clean.
